# Usage
|               Keys               | Function                                                  |
|:--------------------------------:|-----------------------------------------------------------|
| `h` `j` `k` `l` / `w` `a` `s` `d`| Navigate                                                  |
|                `q`               | Open Settings                                             |
|                `e`               | Return to file manager                                    |
|              `space`             | (Only available in settings) activate/deactivate settings |
